Uso URL's absoluta & relativa

Vou testar agora o uso de URL absoluta e URL Relativa, vejamos:

  • Site Generation Brasil Essa primeira URl é chamada de ABSOOLUTA porque o caminho HTTP é fornecido!
  • Cursos Essa segunda URL é chamada de RELATIVA porque te apresenta a pasta ou página contida no site. Nesse caso, ao clicar o usário seria levado para a pasta/página de cursos oferecidos. Neste caso seria a '/#picker' que leva o usuário aos programas de formação da Gen.
  • E-mail para contato a tag a pode ser usada também para informar um endereço de e-mail. Ao clicar o navegador vai buscar um aplicativo de e-mail instalado no computador do usuário já com o campo de e-mail preenchido para envio.
